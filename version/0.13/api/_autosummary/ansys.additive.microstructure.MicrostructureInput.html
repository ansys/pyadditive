Input parameters for microstructure simulation.

Units are SI (m, kg, s, K) unless otherwise noted.

Methods

Attributes

MicrostructureInput.cooling_rate

Material cooling rate (K/s).

MicrostructureInput.id

User provided identifier for this simulation.

MicrostructureInput.machine

Machine related parameters.

MicrostructureInput.material

Material used during simulation.

MicrostructureInput.melt_pool_depth

Melt pool depth (m).

MicrostructureInput.melt_pool_width

Melt pool width (m).

MicrostructureInput.random_seed

Random seed for the simulation.

MicrostructureInput.sample_min_x

Minimum x coordinate of the geometry sample (m).

MicrostructureInput.sample_min_y

Minimum y coordinate of the geometry sample (m).

MicrostructureInput.sample_min_z

Minimum z coordinate of the geometry sample (m).

MicrostructureInput.sample_size_x

Size of the geometry sample in the x direction (m).

MicrostructureInput.sample_size_y

Size of the geometry sample in the y direction (m).

MicrostructureInput.sample_size_z

Size of the geometry sample in the z direction (m).

MicrostructureInput.sensor_dimension

Dimension of the sensor (m).

MicrostructureInput.thermal_gradient

Material thermal gradient (K/m).

MicrostructureInput.use_provided_thermal_parameters

If True, indicates that cooling_rate, thermal_gradient, melt_pool_depth and melt_pool_width have been provided by the user.
