property microstructure.use_provided_thermal_parameters: bool#

Flag indicating if the cooling_rate, thermal_gradient, melt_pool_depth, and melt_pool_width parameters have been provided by the user.

If the value is False, these values will be calculated. Default is False.

property microstructure.cooling_rate: float#

Material cooling rate (K/s).

Valid values are from 1e5 to 1e7.

property microstructure.thermal_gradient: float#

Material thermal gradient (K/m).

Valid values are from 1e5 to 1e8.

property microstructure.melt_pool_width: float#

Melt pool width (m).

This is the width of the melt pool measured at the top of the powder layer, which corresponds to the WIDTH value in MeltPoolColumnNames class.

Valid values are from 7.5e-5 to 8e-4.

property microstructure.melt_pool_depth: float#

Melt pool depth (m).

This is the depth of the melt pool as measured from the top of the powder layer, which corresponds to the DEPTH value in the MeltPoolColumnNames class.

Valid values are from 1.5e-5 to 8e-4.

property microstructure.random_seed: int#

Random seed for the simulation.

Valid values are from 1 to 4294967295.
