ThermalPropertiesDataPoint#

class ThermalPropertiesDataPoint(*, density: float = 0, density_ratio: float = 0, specific_heat: float = 0, specific_heat_ratio: float = 0, temperature: float = 0, thermal_conductivity: float = 0, thermal_conductivity_ratio: float = 0)#

Provides the container for temperature-dependent properties.
