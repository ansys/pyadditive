.. py:function:: create_channel(target: str, transport_mode: ansys.additive.core.server_connection.constants.TransportMode | str, certs_dir: pathlib.Path | str | None, uds_dir: pathlib.Path | str | None, uds_id: str | None, allow_remote_host: bool = False, max_rcv_msg_len: int = MAX_RCV_MSG_LEN)

   
   Create a gRPC channel.


   :Parameters:

       **target: str**
           IP address of the host to connect to, of the form ``host:port``.

       **transport_mode** : :obj:`TransportMode` | :class:`python:str`
           The transport mode to use for the connection. Can be a member of the :class:`TransportMode <.constants.TransportMode>` enum or a string
           ('insecure', 'mtls', or 'uds').

       **certs_dir** : :obj:`Path` | :class:`python:str` | :data:`python:None`
           Directory containing certificates for mTLS connections. Required if `transport_mode` is 'mtls'.

       **uds_dir** : :obj:`Path` | :class:`python:str` | :data:`python:None`
           Directory containing Unix Domain Socket files. Required if `transport_mode` is 'uds'.

       **uds_id** : :class:`python:str` | :data:`python:None`
           Identifier for the Unix Domain Socket. Required if `transport_mode` is 'uds'.

       **allow_remote_host: bool**
           Whether to allow connections to remote hosts when using 'insecure' or 'mtls' transport modes.

       **max_rcv_msg_len: int**
           Size, in bytes, of the buffer used to receive messages. Default is :obj:`MAX_RCV_MSG_LEN`.



   :Returns:

       channel: :obj:`grpc.Channel`
           Insecure or secure gRPC channel, depending on the transport mode.

       uds_file: :obj:`Path` | :data:`python:None`
           Path to the Unix Domain Socket file if using 'uds' transport mode, otherwise None.




   :Raises:

       :obj:`ValueError`
           If the target string is improperly formed, the transport mode is invalid, or an unsupported transport mode is specified.

       :obj:`ConnectionError`
           If unable to connect to the Unix Domain Socket when using 'uds' transport mode.
